Tag input dengan atribut type="text"

Ini text box dengan nama teks ku ukuran 16 chr dan maksimal karakter yang dapat di tampung 20 chr(character)

Tag input dengan atribut type="password"
Ini passwordbox dengan nama password_ku ukuran 16 chr dan maksimal karakrer yang dapat ditampung 20 chr(character).

Tag input dengan atribut type="radio"
Radio 1 --> nama radio_ku isi/value isi_1
Radio 2 --> nama radio_ku isi/value isi_2
Jika nama radio tidak sama maka tidak di anggap satu pilihan. Untuk membktikannya silahkan ganti salah satu nama dari radio di atas untuk mengetahui perbedaanya.

Tag input dengan atribut type="checkbox"
Checkbox 1 --> nama checkbox_ku_1 isi/value isi_1
Checkbox 2 --> nama checkbox_ku_2 isi/value isi_2

Tag input dengan atribut type="submit"

Tombol ini berguna untuk mem-post(mengirimkan variabel dan isi variabel yang ada pada form ke file tujuan(action="file tujuan"))

Tag input dengan atribut="reset"

Tombol ini berguna untuk membersihkan seluruh tulisan yang ada pada seluruh isian form. Coba isikan semua textbox dan radio lalu tekan tombol ini.

Tag input dengan atribut type="hidden"
Tipe ini berguna Jika si webmaster menginginkan variabel yang sifatnya sudah di definisikan. dan tidak menampilkan output apapun.

Tag input dengan properti type="image"
height="98" width="98"
image memiliki fungsi yang sama dengan type submit
